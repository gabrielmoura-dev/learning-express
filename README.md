# 3.1 Express Server

Servidor básico criado com **Node.js** e **Express.js** como parte dos estudos do Bootcamp Full-Stack da Dr. Angela Yu.

## 🚀 Tecnologias

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)

## 📦 Instalação

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/gabrielmoura-dev/learning-express.git
cd learning-express/3.1\ Express\ Server
npm install
```

## ▶️ Como rodar

```bash
node index.js
```

O servidor estará rodando em `http://localhost:3000`

## 📁 Estrutura

```
3.1 Express Server/
├── index.js        # Entrada da aplicação
├── package.json    # Dependências e configurações

```

## 📝 Sobre

Este projeto é parte de uma série de estudos práticos em desenvolvimento back-end, explorando conceitos de servidores HTTP, rotas e APIs REST com Express.
